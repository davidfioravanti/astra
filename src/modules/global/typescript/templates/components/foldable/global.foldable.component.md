## Introduction

The foldable component emulates the functionality of the HTML-native `<details>` and `<summary>` tags while providing extra options and features. The component is comprised of two main elements: a control/toggle mechanism and the container whose visibility it controls. The content inside of that container element is passed in via the `children` property just like any other React component.
